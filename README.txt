# LangChain Building Blocks

This repository explains the **core building blocks of LangChain** using simple and practical Python examples.

The goal of this project is to help beginners understand how **LangChain works internally** before moving to more advanced topics such as **Retrieval Augmented Generation (RAG)** and **AI Agents**.

---

## Prerequisites

Before exploring this repository, you should have:

* Basic understanding of **Python programming**
* Basic idea of how APIs work (optional but helpful)

This repository assumes that **basic Python concepts are known**.
